# Screenshots

Marketing images for [README.md](../../README.md) and [index.html](../index.html). Updated for **ThatCode v2.7.1**.

| File | Use |
|------|-----|
| `chat.png` | Main view — onboarding, quick prompts, input bar |

| `connections.png` | Settings → Connection (API URL, key, model presets) |
| `workspace.png` | Settings → Workspace (project folder, verify loop) |

## Replacing captures

1. Run `npm run tauri dev` (or a release build).
2. Capture at **1280×720** or **1440×900**.
3. Overwrite the files above (keep filenames so README links stay valid).
4. Optimize PNG/GIF size before commit (keep large assets under ~5 MB for GitHub).
